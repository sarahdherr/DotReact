# DotReact

### Run DotReact Locally
In terminal:
1. `git clone https://github.com/sarahdherr/DotReact.git`
2. `cd DotReact`
3. `npm install`
4. `npm run build`
5. `npm start`
Then, open https://localhost:8080

Versions I ran it on:
* node: v8.4.0
* npm: 5.3.0

To run full test suite:
1. Make sure you have `npm install`ed
2. `npm test`
